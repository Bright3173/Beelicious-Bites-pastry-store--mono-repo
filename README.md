# Beelicious Bite 

Beelicious Bite is a modern full-stack e-commerce platform built for a pastry and bakery business.
The application allows customers to browse products, add items to their cart, place orders, and manage their accounts,
while administrators can manage products through a secure admin dashboard.

## Features

### Customer Features

* User registration and authentication using JWT
* Secure login and logout functionality
* Forgot password and OTP-based password reset
* Browse products by category
* Product details and pricing display
* Shopping cart management
* Responsive design for desktop and mobile devices
* Secure payment gateway

### Admin Features

* Secure admin-only access
* Create new products
* Edit existing products
* Delete products
* Upload product images using Cloudinary
* Manage product inventory and pricing
* Protected routes and role-based authorization

## Tech Stack

### Frontend

* Next.js
* React
* TypeScript
* Redux Toolkit
* Formik
* Yup Validation
* React Bootstrap
* Axios

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication
* Bcrypt Password Hashing
* Multer
* Cloudinary

## Authentication & Security

The application implements secure authentication and authorization using:

* JWT Access Tokens
* Refresh Tokens
* Password Hashing with Bcrypt
* Role-Based Access Control (RBAC)
* Protected API Routes
* Admin Authorization Middleware

## Image Management

Product images are uploaded and stored using Cloudinary, providing:

* Secure cloud storage
* Fast image delivery
* Optimized image performance
* Scalable media management

## Project Goal

The goal of Beelicious Bite is to provide a seamless online shopping experience for bakery customers while offering
administrators an easy-to-use dashboard for managing products and inventory.

This project demonstrates full-stack development skills including frontend development, backend API design, authentication,
database management, cloud media storage, and role-based access control.
